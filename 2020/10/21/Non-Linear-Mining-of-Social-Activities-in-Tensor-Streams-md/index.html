Non-Linear Mining of Social Activities in Tensor Streams

发表于 2020-10-21 更新于 2020-10-28 分类于 论文笔记 阅读次数： Valine：
  • 来源： KDD-2020

    摘要

      给出如谷歌网页搜索日志的长时时间序列，我们应如何精确的预测他们的未来活动？
      文章提出了名为CUBECAST的流方法，用于捕捉张量流中基础的的趋势和季节性以及抽取动态变化中的多维关系。此方法有如下属性：(a)有效性：发现了趋势和季节性，并将它们的动态归纳为同时存在的非线性潜在空间；(b)自动性：不需要任何的参数调节和先验知识就可以自动的识别和建模结构化的模式；(c)扩展性：对一个半无限的张量流收集，其增量和自适应的检测模式的移动点；

    文章内容

    引言

      预测张量流有两个难题：(a)可观察数据背后的多重因素；(b)随时间变化的模式：不仅要理解趋势而且要理解其动态的变化；
      CUBECAST：可以同时捕获时间演变趋势和季节性以及张量流中的多个离散模式。
  1. 长时趋势：figure1(a)展示了文章算法，它扫描最新张量的增量和捕捉非线性趋势，以实现长期预测。
  2. 季节性：figure1(b)展示了从张量流中提取的季节特征。
  3. 多元的动态模式：figure1(c)展示了06-08年五十个国家的CUBECAST聚类结果

  1. figure2(a)将美国和英国分为一组因为它们有相似的动态性，包括对“gap”的强季节性
  2. figure2(b)显示了“h&m”搜索计数在意大利大幅增长的情况下搜索计数时刻之间的模式差异。通过切换一个即将到来的模式模型，它有效地预测张量流。

相关工作

  时间序列的传统方法如：AR、linear dynamical systems(LDS)、kalman filter(KF)及其扩展等；RegimeCast和OrbitMap是实时预报方法，可以捕捉多维事件流中的潜在非线性动态性，但是这些方法不能在流中建模季节性模式；LSTM、GRUs；
  对于挖掘高阶数据来说，张量分解是一个理解数据潜在因素的有力技术；AutoCyclone提供了稳健的因子分解，将季节叠加时间序列视为张量，可用于从时间序列中分离基本趋势、季节性和异常值。CompCube、PowerCast；

CUBECAST

  • 三个能力：非线性潜在动态性；季节性；张量流的共同演化模式；
  • 单一位置潜在的非线性动态性
    • 活动的时间依赖性可以描述为
      $z_{t+1}=Az_{t}+Bz_{t}\bigotimes z_{t}$
      $e_{t}=Wz_{t}$
      其中$\bigotimes$表示两个向量外积；
  • 潜在的季节动态性
    • 通过扩展基础模型来建模季节/周期模式，具体地，作者希望为季节性定义另一种潜在空间可以随时间与线性和非线性活动进行交互。
      $\begin{bmatrix}
      z_{t+1}\\
      v_{t+1}
      \end{bmatrix}=A\begin{bmatrix}
      z_{t}\\
      v_{t}
      \end{bmatrix}+B\begin{bmatrix}
      z_{t}\\
      v_{t}
      \end{bmatrix}\bigotimes\begin{bmatrix}
      z_{t}\\
      v_{t}
      \end{bmatrix}$
      $e_{t}=Wz_{t}+U(v_{t}\bigcirc S_{t\quad mod\quad p})$
      其中$\bigcirc$表示向量对应位置元素的乘积
  • 有多个位置的完整模型
    • 假设不同的位置有不同的活动，我们应当如何描述在庞大张量流中随时间变化的系统
    • Definition
      • single regime parameter set
      • full parameter set
      • regime assignment set

        算法优化

  • 有效和精确的预测未来模型仍然有两个问题：(a)当自适应的产生和切换体制时能够及时的预测未来事件(b)自动挖掘和估计多元非线性动态性
